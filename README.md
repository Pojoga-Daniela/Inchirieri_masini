# Aplicație pentru închirieri mașini

Această aplicație este destinată unui birou de închirieri auto și permite gestionarea mașinilor disponibile și a închirierilor realizate de clienți. Accesul în aplicație se face pe baza autentificării angajaților. Utilizatorul poate vizualiza lista mașinilor firmei, verifica disponibilitatea acestora și înregistra o închiriere. Pentru fiecare client se vor introduce date precum nume, prenume, CNP și perioada de închiriere (data de început și data de sfârșit). Informațiile vor fi salvate într-o bază de date, iar mașina închiriată va deveni indisponibilă pentru alți clienți în perioada respectivă.
